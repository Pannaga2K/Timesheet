# TIMESHEET
    - Basic timesheet for maintaining attendance of users by using CRUD operations with respect to MySQL.

## PACKAGES USED

```
    - FRONTEND
        - react-router-dom
        - axios
    - BACKEND
        - express
        - mysql
        - body-parser
        - cors
```

## USAGE

```
    - FRONTEND
        - cd FRONTEND/timesheet
        - npm start
    - BACKEND
        - cd BACKEND
        - nodemon index.js
```
